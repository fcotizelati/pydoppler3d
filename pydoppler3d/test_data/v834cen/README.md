# V834 Cen Sample Data

This directory contains the V834 Cen input spectra distributed with the
`doptomog` archive at CDS entry `J/A+A/595/A47`.

Source:

* Kotze, Potter, & McBride 2016, A&A, 595, A47
* CDS archive: https://cdsarc.cds.unistra.fr/ftp/J/A+A/595/A47/

The spectra are the `sources/mcv/in/` example data from `doptomog-2.0.tar.gz`.
The file `spectra/mcv` lists the 59 spectra and orbital phases. The individual
spectra in `spectra/mcv*.txt` contain wavelength and flux columns. The bundled
configuration files specify the He II 4686 line extraction used by the example.

These files are third-party research data bundled for examples and tests. They
are not relicensed as PyDoppler3D source code; see the repository `NOTICE` file
for citation and redistribution notes.
